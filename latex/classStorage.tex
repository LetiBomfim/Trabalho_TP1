\doxysubsection{Detailed Description}
Serviço de armazenamento de dados do sistema. 

Responsável por gerenciar a persistência e o acesso aos dados de contas, carteiras e ordens de investimento. Fornece métodos para adicionar, remover, buscar e listar Implementa o padrão Singleton para garantir uma única instância durante a execução do sistema.
