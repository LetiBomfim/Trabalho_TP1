\doxysubsection{Detailed Description}
Entidade que representa uma transacao financeira. 

Contem todos os dados necessarios para\+:
\begin{DoxyItemize}
\item Identificacao do ativo (codigo de negociacao);
\item Valor e quantidade da operacao;
\item \doxylink{class_data}{Data} da transacao;
\item Vinculo com a carteira; 
\end{DoxyItemize}
